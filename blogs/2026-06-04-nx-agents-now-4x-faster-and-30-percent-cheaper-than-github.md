Nx Cloud released optimized resource classes and improvements to task distribution through Continuous Assignment, achieving 74% reduction in wall-clock time and 30% cost savings per run compared to GitHub Actions in large monorepos. The improvements leverage Nx's architectural advantage of understanding the complete task pipeline, allowing dynamic work distribution that keeps compute resources continuously utilized rather than idle. These results were benchmarked against equivalent GitHub Actions runners in real-world monorepo scenarios.
